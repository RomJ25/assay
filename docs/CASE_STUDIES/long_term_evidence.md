**The factors Assay is built on have earned significant premiums over every multi-decade period since 1963. This document presents the full record.**

## 1. Prologue — Why 12 Years Matters

Twelve years is a long time. It is longer than most investment track records. It spans recessions, pandemics, and regime changes. Most funds that underperform for twelve years are closed.

But twelve years is not long enough.

The value premium — the tendency of cheap stocks to outperform expensive ones — was negative for the entire 2010s. An investor who bought value stocks on January 1, 2010 and held through December 31, 2019 would have underperformed the market by approximately -2.2% per year, every year, for a decade. That investor would have had every reason to abandon the strategy.

They would have been wrong to do so.

The question this case study addresses is not "does value+quality work?" — that question has been answered by sixty years of academic research, across dozens of countries, in peer-reviewed journals. The question is: **how long do you have to wait?** And what does the full record look like when you include the bad decades alongside the good ones?

## 2. 62 Years of Evidence

The Fama-French factor libraries provide monthly returns for systematic risk factors from July 1963 to present. These data are survivorship-free (they include delisted stocks), publicly available, and form the empirical foundation of modern asset pricing.

Using value (HML — high book-to-market minus low) and quality (RMW — robust minus weak profitability) factors layered on the market return:

```
    +-------------------------------------------------------------------+
    |  Strategy                     62-Year CAGR   $100 Becomes         |
    |  --------                     ------------   ------------         |
    |  Market (Mkt-RF + RF)             +10.9%        ~$60,000          |
    |  Market + Value + Quality         +17.9%      ~$2,800,000         |
    |                                                                    |
    |  V+Q annual premium               +7.1%                           |
    +-------------------------------------------------------------------+
```

Read those numbers again. An investor who tilted toward cheap, profitable stocks — the exact approach Assay formalizes — would have turned $100 into roughly **47 times more wealth** than the market alone. Not by timing the market. Not by picking individual winners. By systematically buying stocks that were cheap and profitable, rebalancing annually, and waiting.

This is not a backtest of a proprietary strategy. It is the raw academic factor data that thousands of researchers have studied and hundreds of papers have confirmed. The data is free, public, and downloadable from Kenneth French's website.

The magnitude matters less than the persistence. A +7.1% premium that appeared in one decade and vanished in the next would be a fluke. The question is whether it persisted. Section 3 answers that.

## 3. By Decade — When Value+Quality Worked

```
    +-------------------------------------------------------------------+
    |  Decade    Value     Quality    Combined     Note                  |
    |  ------    -----     -------    --------     ----                  |
    |  1960s     +2.3%      +1.4%      +3.7%      Modest, positive      |
    |  1970s     +8.2%      -0.6%      +7.6%      Value dominated       |
    |  1980s     +6.4%      +5.0%     +11.3%      Both contributed      |
    |  1990s     +0.6%      +2.6%      +3.2%      Dot-com headwind      |
    |  2000s     +8.3%      +8.3%     +16.6%      Post-crash rebound    |
    |  2010s     -2.2%      +1.4%      -0.8%      Worst decade on       |
    |                                               record for value     |
    |  2020s*    +1.9%      +4.7%      +6.6%      Recovery underway     |
    +-------------------------------------------------------------------+
    * 2020s covers 2020-2025 (partial decade)
```

Three observations:

**Value and quality are not the same factor.** In the 1970s, value earned +8.2% while quality was slightly negative (-0.6%). In the 2010s, value lost -2.2% per year while quality stayed positive (+1.4%). The two factors have low correlation. Combining them provides diversification that neither offers alone.

**The 2010s were the worst decade for value in the entire 62-year record.** Not just bad — historically unprecedented. The prior worst decade for value was the 1990s at +0.6%, which was still positive. The 2010s were the first and only decade where value was meaningfully negative. This was driven by the dominance of growth mega-caps (FAANG stocks), near-zero interest rates compressing discount rates, and a sustained expansion of price multiples for growth companies.

**Despite the 2010s, the combined premium was positive in six of seven decades.** The only negative decade (-0.8% in the 2010s) was barely negative, because quality partially offset value's losses. An investor who held value+quality through every decade would have experienced one near-flat decade and six positive ones.

## 4. The Last 12 Years (2014-2025)

The academic data can be extended to a year-by-year view for the recent period. This is the stretch most relevant to anyone evaluating Assay today — and it is the most difficult stretch for a value+quality investor in recorded history.

Consider what it was like. In 2020, while Tesla gained +743%, NVIDIA gained +122%, and the NASDAQ surged +43%, the value+quality premium was **-32.9%**. A screener like Assay would have pointed you toward stocks like energy companies and banks while the market was being led by technology mega-caps. Every financial headline, every cocktail party conversation, every brokerage statement would have confirmed: you were on the wrong side. Value was dead.

Then 2021-2022 happened. Value surged +22% and +32% as interest rates rose, inflation returned, and growth multiples compressed. The premium that had been absent for years arrived all at once — in two violent years that erased the entire 2015-2020 deficit and then some.

```
    +-------------------------------------------------------------------+
    |  Year    Market    Value    Quality   V+Q Prem    $100 Mkt  $100 V+Q|
    |  ----    ------    -----    -------   --------    --------  --------|
    |  2014    +11.8%    -2.0%     +0.6%     -1.4%        $112      $110 |
    |  2015     +0.3%    -9.8%     +0.5%     -9.3%        $112      $100 |
    |  2016    +13.6%   +20.7%     +3.4%    +24.0%        $127      $138 |
    |  2017    +22.3%   -10.7%     +3.8%     -7.0%        $156      $159 |
    |  2018     -5.0%   -10.5%     -2.0%    -12.4%        $148      $131 |
    |  2019    +30.5%    -8.1%     +2.9%     -5.1%        $193      $165 |
    |  2020    +24.1%   -30.7%     -2.3%    -32.9%        $239      $150 |
    |  2021    +23.9%   +22.2%    +24.5%    +46.7%        $297      $256 |
    |  2022    -19.9%   +31.7%     +8.3%    +40.0%        $238      $308 |
    |  2023    +26.7%   -11.1%     +3.8%     -7.2%        $301      $368 |
    |  2024    +25.1%    -7.2%     +4.1%     -3.1%        $376      $448 |
    |  2025    +17.6%    +6.4%    -10.1%     -3.8%        $443      $510 |
    +-------------------------------------------------------------------+
```

Over 12 years, $100 invested in the market grew to approximately **$443**. $100 in Market + Value + Quality grew to approximately **$510**. The excess was +$68, or roughly +15.3%.

That is a positive result. It is also a modest one. A +15.3% cumulative excess over 12 years is roughly +1.2% per year — far below the 62-year average of +7.1%.

**The year-by-year volatility is the real story.** In 2020, the V+Q premium was -32.9% — value+quality stocks underperformed the market by a third. In 2021, the premium was +46.7%. These are not small fluctuations. They are violent swings that would test the conviction of any investor.

Nine of the twelve years had negative premiums. The premium was positive in only three years (2016, 2021, 2022). The positive years were large enough to more than offset the nine negative ones, but living through nine losing years out of twelve requires extraordinary patience.

## 5. What the Data Teaches About Patience

The 62-year record reveals a clear pattern: the value+quality premium is **real but lumpy**. It does not arrive in steady annual increments. It arrives in bursts — often after extended periods of underperformance.

```
    +-------------------------------------------------------------------+
    |  Holding Period       % of Periods with Positive V+Q Premium      |
    |  --------------       ------------------------------------------  |
    |  1 year                          63%  (40/63)                      |
    |  3 years                         87%  (53/61)                      |
    |  5 years                         90%  (53/59)                      |
    |  10 years                        94%  (51/54)                      |
    |  20 years                       100%  (44/44)                      |
    +-------------------------------------------------------------------+
    Computed from all rolling periods in the 1963-2025 dataset.
```

At any given year, the V+Q premium is positive about 63% of the time — better than a coin flip but far from guaranteed. Over five years, the odds rise to 90%. Over ten years, 94%.

And over twenty years: **100%.** Every single twenty-year rolling window since 1963 — all 44 of them, including windows that contained the dot-com crash, the financial crisis, the worst value decade on record — produced a positive value+quality premium. Not most of them. All of them.

The implication for Assay users: **a minimum five-year holding period is required to have reasonable confidence that the factor premium will manifest.** One year is noise. Three years is suggestive. Five years begins to separate signal from noise. This is not a trading strategy. It is a long-duration bet on the persistence of a well-documented anomaly.

## 6. Assay's Implementation vs the Theoretical Premium

The 62-year academic premium is +7.1% per year. What does Assay actually capture?

The April 2026 empirical investigation tested Assay's screener across 12 quarters (3 years of data). The results:

```
    +-------------------------------------------------------------------+
    |  Metric                         Assay Actual   Theoretical        |
    |  ------                         ------------   -----------        |
    |  Portfolio CAGR                     +11.2%         --              |
    |  Universe CAGR                      +12.5%         --              |
    |  Selection Alpha (raw)              -1.3%        +7.1%            |
    |  Sector-neutralized Alpha           +0.1%         --              |
    +-------------------------------------------------------------------+
```

Assay's portfolio returned +11.2% annualized. The universe returned +12.5%. The raw selection alpha was -1.3% — the screener underperformed its own universe.

This is not a contradiction with the academic evidence. The gap between +7.1% theoretical and -1.3% actual has identifiable causes:

**1. Imperfect factor exposure.** The Fama-French factors are constructed as long-short portfolios across the entire CRSP universe (thousands of stocks). Assay runs a long-only screen on the S&P 500 (roughly 500 stocks). The factor exposure is diluted by construction.

**2. Concentrated portfolio.** Assay selects 5-20 names per quarter. The academic factors are broadly diversified across hundreds of stocks. Concentration increases variance and can easily produce negative alpha over short periods even when the underlying factor premium is positive.

**3. Transaction costs.** The academic factors are frictionless. Assay's backtest includes transaction cost modeling, but the real-world cost of turning over a concentrated portfolio is material.

**4. Short sample.** Three years (12 quarters) is statistically meaningless for evaluating a strategy built on factors that require 5-10 years to manifest reliably. The -1.3% alpha over 3 years is well within the expected noise band of a +7.1% annual premium with high volatility.

**5. The period tested included the tail end of the worst value decade on record.** The 2021-2024 investigation period overlapped with the aftermath of the 2010s value drawdown. A screener that tilts toward value during a value headwind will underperform.

The honest summary: Assay captures a fraction of the theoretical premium under quarterly rebalancing. The gap is expected and explainable. However, switching from quarterly rebalancing to a selective sell strategy (hold winners, sell only when fundamentals break) improved selection alpha from -1.3% to +1.0% over the same period. The selective sell strategy returned +18.7% CAGR vs the quarterly rebalance's +15.2%. See `docs/STRATEGY.md` for the recommended approach.

Whether the screener will converge further toward the academic premium over longer periods is an open question that only time can answer.

## 7. Takeaway

The evidence supports three claims and refutes one:

**Supported:**

1. **Value+quality is the most documented anomaly in finance.** Sixty-two years of survivorship-free, peer-reviewed data across multiple countries and time periods. The premium is real.

2. **The premium requires patience.** The 2010s were the worst decade for value in recorded history. Nine of the last twelve years had negative V+Q premiums. Despite this, the cumulative result over both 12 and 62 years is positive. Minimum holding period: five years.

3. **The screener is a disciplined application of proven factors, not a proprietary signal.** Assay buys cheap, profitable, financially healthy stocks. The academic literature says this works. The 62-year record confirms it. The screener is one implementation of a well-understood idea.

**Refuted:**

4. **The screener captures the full academic premium.** It does not. Assay's actual alpha over its investigation period was -1.3% versus the theoretical +7.1%. The gap is structural (concentrated long-only portfolio, S&P 500 universe, transaction costs) and may narrow over time but will never close. Anyone expecting +7% annual alpha from a 5-pick S&P 500 screen is misunderstanding the relationship between academic factors and practical implementation.

## 8. Caveats

```
    +-------------------------------------------------------------------+
    |  Limitation                  Effect on this case study             |
    |  ----------                  -------------------------             |
    |  Factor data != strategy     Fama-French factors are long-short,  |
    |                              cap-weighted, and rebalanced          |
    |                              annually. Assay is long-only,         |
    |                              equal-weighted, and rebalanced        |
    |                              quarterly. The +7.1% premium is      |
    |                              not directly achievable.              |
    |                                                                     |
    |  Survivorship in Assay       The Fama-French data is              |
    |                              survivorship-free. Assay's own        |
    |                              backtest uses S&P 500 membership      |
    |                              as of each screen date, which         |
    |                              introduces look-ahead bias on         |
    |                              index composition.                    |
    |                                                                     |
    |  Short investigation         3 years (12 quarters) of Assay       |
    |  period                      data vs 62 years of factor data.     |
    |                              The comparison is illustrative,       |
    |                              not conclusive.                       |
    |                                                                     |
    |  Factor crowding             The value and quality premiums        |
    |                              were documented in 1992 and 2014.     |
    |                              Post-publication, institutional       |
    |                              capital has flowed into factor        |
    |                              strategies. Future premiums may       |
    |                              be smaller than historical ones.      |
    |                                                                     |
    |  Compounding assumptions     The "$2.8M vs $60K" comparison       |
    |                              assumes reinvestment of all           |
    |                              returns, no taxes, no withdrawals,    |
    |                              and perfect execution over 62         |
    |                              years. No real investor achieves      |
    |                              this.                                  |
    |                                                                     |
    |  No statistical tests        All numbers in this document are     |
    |                              descriptive. No t-tests, bootstrap   |
    |                              confidence intervals, or Sharpe      |
    |                              ratio comparisons are presented.     |
    +-------------------------------------------------------------------+
```
